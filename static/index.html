gsky

gsky, pronounced [jee-skee], is a distributed geospatial data server exposing OGC services developed at the NCI.

In a few lines

gsky is designed to:

  • Scale: Distributed solution that scales in serving volume of data and concurrent users
  • Work with raw data: Transparent ingestion of geospatial collections in many scientific data formats such as NetCDF, HDF or geoTIFF
  • Compute on-the-fly: No caching, no precomputed products. Resampling and reprojection happen in real time
  • Standards: Implements OGC standards such as WMS and WPS
  • More functionalities to come….

Want to try it?

Any OGC compatible client can be used. For example:

Go to the National Map web site

http://nationalmap.gov.au

Click on Add Data -> My Data -> Web Data and type

http://gsky.nci.org.au/ows
